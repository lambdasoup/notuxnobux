Welcome to No Tux No Bux!

#notuxnobux aims at game publishers aswell as Linux gamers.

Publishers

If your game is not available for Linux, you will not get any money from gamers which follow #notuxnobux philosophy.

Gamers

Do not buy a game as long as there is not native port. Resist the urge to buy the game and boot into other OSs to play it right away.

About this site

This project is obviously raw and needs work. You can help speed up development at notuxnobux on GitHub.
